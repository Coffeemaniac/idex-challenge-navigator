DISC 14 Indian Navy DISC14-38 PS 38

Problem Statement

The project aims to develop a standalone software that undertakes processing, identification and classification of recorded data of sonobuoys dropped by P8I

Challenge Brief / Definition

The project aims to develop a software capable of following:
· Automatic running of recorded sonobuoy data
(waterfall display) to undertake detection of fundamental frequency along with associated harmonics.
· Loading of LOFAR database to undertake automatic co-relation of discretes with the library.
· Presentation of bearing of all discretes recorded for easy co-relation of the tactical picture.
· Processing of multiple buoy data simultaneously in batches of 2,4,8 and 16.

Technology Domains

Suitable algorithm to detect and classify data

Application / Use Case

The system is envisaged to be used for post mission analysis of sonobuoy data obtained from P8I

Project Outcome

The project aims to develop a software that can be deployed along with aircrew for ground analysis at air station and at detached locations.

Expectation from Prototype

Technology analysis of sonobuoy data to facilitate aircrew with pre- mission briefs for ASW sorties.
