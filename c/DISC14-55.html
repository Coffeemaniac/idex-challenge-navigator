4. High speed and ultra-realistic graphics to be used to create the virtual world. 50 different missions (each for CCT and
TAC) to be preferred in the software, that should be user selectable.
5. Large size video wall for display of first person/ third person view, mission replay, flight data and user selectable VR/MR picture of any individual selected, along with high end acoustic speakers.
6. Required display terminals, monitors, processors, servers, and other hardware as required.
7. User evaluation will be conducted at every stage of development. Missions will be loaded on user requirements with varied levels of difficulty, as envisaged.
8. Detailed user specifications will be shared at the time of development by the winner. Integration of aircraft parameters will be done at / with the assistance from SDI, Bengaluru.
9. Final evaluation will be conducted once all aspects related to user requirement are integrated in the system.
First set of simulator system will be set up at GRTC for functional evaluation. Upgradation, wherever necessary is to be carried out and re-tested on-site.
